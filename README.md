# candytown-mfa

Dynamic MFA exercise with building materials in Candy Town



This is a hands-on set of activities for college-level learners to:



* Practice MFA mass balance formula
* See how dynamic MFAs are created from the bottom-up
* Learn about lifespan distributions and stocks
* Recognize that materials eventually become waste
* Consider opportunities presented by this data
* Observe challenges with matching data across stakeholders



The latest files are v4. Included are handouts (docx and pdf), spreadsheets (solved, unsolved), and overview slides.



This activity was developed in 2022 and has been completed by at least four sets of graduate student at various universities.  



There is always room for improvement; you are encouraged to fork these files and share your edits on GitHub!
